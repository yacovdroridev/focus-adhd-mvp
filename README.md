# Focus Field — ADHD Music MVP

A tiny browser-only MVP that generates procedural focus audio for ADHD-friendly work sessions.

## Run

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 4173
```

Then visit http://localhost:4173

## MVP features

- Procedural ambient pads
- Brown/pink-ish noise bed
- Soft rhythmic pulse
- Optional binaural drift
- ADHD stimulation dial
- 15/25/45/90 minute sessions
- No backend, no dependencies, no tracking
